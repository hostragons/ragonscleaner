# 🚀 RagonsCleaner

Professional cPanel server cleanup tool by **Hostragons Global Limited**

---

## 🧠 What is RagonsCleaner?

RagonsCleaner is a powerful, safe and production-ready Bash script designed to free up disk space on cPanel servers.

It removes unnecessary files such as:

* cPanel backups
* Trash directories
* Softaculous backups
* WordPress backups
* Temporary files
* Old logs
* Mail spam & trash

---

## ⚡ Features

* ✅ Colorful and clean CLI output
* ✅ Step-by-step cleanup process
* ✅ Shows freed disk space per step
* ✅ Displays disk usage before & after
* ✅ Calculates total freed space
* ✅ Production-safe logic
* ✅ No impact on active websites or databases

## 📦 Installation

```bash
git clone https://github.com/YOUR_USERNAME/ragonscleaner.git
cd ragonscleaner
chmod +x ragonscleaner
```

---

## ▶️ Usage

```bash
./ragonscleaner
```

Or install globally:

```bash
mv ragonscleaner /usr/local/bin/
ragonscleaner
```

---

## ⚠️ Warning

This script performs **permanent deletions**.

* No undo
* No recycle bin
* Use with caution
